Project Description

Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:NickOsipov/prod-code-lesson.git
    cd prod-code-lesson
    ```
2. Create a project:
    ```bash
    uv python pin 3.11.4
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
